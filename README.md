![Status](https://img.shields.io/badge/Status-Active-brightgreen)

🚀 **Library Management System (LMS) Using Angular** is a **web application** designed to manage library books, users, and borrowing operations efficiently.  
👨‍💻 The project includes **Admin and User login**. Admin can manage books and users, while users can borrow and view books.  
📖 The system demonstrates **Angular fundamentals, component-based architecture, event binding, and session handling**.

---

## ✨ Features

### 👨‍💼 Admin Features
- 🔹 Login with secure session handling  
- ➕ **Add new books and users**  
- ✏️ **Update book and user information**  
- ❌ **Delete books and users**  
- 📊 Track borrowed books and availability  

### 👩‍🎓 User Features
- 🔹 Login using unique credentials  
- 📖 Browse and search available books  
- 📝 Borrow books and view borrowing history  
- 🗂️ See personal account details  

### ⚡ Other Features
- 🔄 Session storage ensures **login and data persist after page refresh**  
- 🌐 Responsive design using **Bootstrap and CSS**  
- 🖥️ Real-time UI updates when data changes  
- 🎯 Clean **Angular component-based architecture**  

---

## 🛠️ Technologies Used

- 🔹 **Angular** – Frontend framework for building SPAs  
- 🔹 **TypeScript** – Strongly typed language for Angular components  
- 🔹 **HTML5 / CSS3 / Bootstrap** – Responsive design and layout  
- 🔹 **Local/session storage** – Persisting user session and data  
- 🔹 **Angular CLI** – Project management and build tools

## 📌 How to Run the Project

1️⃣ **Clone the repository**  

```bash
git clone https://github.com/Satasiya-Daksh-Maheshbhai/Library_Management_System_Using_Angular.git

```

2️⃣ **Navigate to the project folder**  

```bash
cd Library_Management_System_Using_Angular
```

3️⃣ **Install dependencies**  

```bash
npm install
```

4️⃣ **Run the Angular development server**  

```bash
ng serve
```

5️⃣ **Open the application**  

```text
http://localhost:4200
```
